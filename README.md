# 🌾 Kisan Digital Sathi

## AI-Powered Digital Agriculture Assistant for Indian Farmers

> “Technology becomes powerful when it reaches the farmer in the simplest way possible.”

---

# 🚀 Overview

Kisan Digital Sathi is an AI-powered AgriTech platform designed to help Indian farmers access:

* 🌱 Crop guidance
* 🌦️ Live weather intelligence
* 📈 Market price information
* 🏛️ Government scheme assistance
* 🌍 Multilingual farming support
* 🤖 AI-powered agricultural recommendations

The project aims to bridge the digital gap between modern agricultural technologies and rural farming communities using Artificial Intelligence.

---

# 🎯 Project Objective

The main objective of Kisan Digital Sathi is to:

* Promote Digital India in agriculture
* Improve farmer digital literacy
* Simplify government scheme access
* Provide AI-based farming guidance
* Support small and marginal farmers
* Build a farmer-friendly smart agriculture ecosystem

# ⚙️ Installation & Setup

# 1️⃣ Clone Repository

```bash
git clone https://github.com/yourusername/Kisan-Digital-Sathi.git
```

---

# 2️⃣ Go to Backend Folder

```bash
cd backend
```

---

# 3️⃣ Install Dependencies

```bash
npm install
```

---

# 4️⃣ Create `.env`

```env
GROQ_API_KEY=your_groq_api_key
WEATHER_API_KEY=your_openweather_api_key
```

---

# 5️⃣ Run Backend Server

```bash
node server.js
```
